If you're an open-source software developer, [Github Sponsors](https://github.com/sponsors) is a way to receive funding directly in Github. However it is lacking several features that make Coindrop a compelling alternative.

In this article, we'll explore some of the downsides of Github Sponsors and how they can be mitigated by supplementing it with Coindrop.

<Image src="/blog-content/github-sponsors-alternative/coindrop-vs-github-sponsors.png" height="407" width="750" alt="Coindrop vs. Github Sponsors" />

To simplify the pros & cons of Github Sponsors vs. Coindrop, we'll compare them in the following categories:
- Payment options
- Transaction fees
- One-time donations

## Payment options

The only payment methods offered for Github Sponsors are Credit Card and PayPal.
Coindrop supports virtually any payment method, including digital wallets (PayPal, CashApp, etc), digital assets & cryptocurrencies (Bitcoin, Ethereum, etc), and subscription platforms (Patreon, Buy Me A Coffee, etc).

## Transactions fees

Github charges a 10% platform fee for organizations to sponsor a developer, and 0% for users
Coindrop has zero platform fees.

## Recurring payments

Github Sponsors only supports recurring donations. No one-time donations.
Coindrop supports virtually any payment platform, many of which have recurring payment options.
