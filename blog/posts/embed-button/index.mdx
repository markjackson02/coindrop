If you're unfamiliar, Coindrop is a **100% free, zero-fee, open-source** way to accept donations. Simply list all payment methods you accept (e.g. PayPal, CashApp, Bitcoin, etc.) and Coindrop provides a simple interface for donors to pay you directly. No middleman involved.

## On what sites can I embed the button?

Any website that supports [HTML](https://www.w3schools.com/html/html_intro.asp) code blocks or [Markdown](https://www.markdownguide.org/).

This includes:
- Website builders like SquareSpace and Wix
- Blog platforms Wordpress and Ghost
- Github README pages
- and many more

<p><u><a href="/" target="_blank">Click here</a></u> to get started. It only takes ~2 minutes to get it up and running.</p>

## Step 3) Paste the code on your website

How to do this will vary based on the website platform you're using. Look for an option to "Add custom HTML", "Add Markdown", or something similar. Paste the code there.

### Wordpress Example

In Wordpress 5.5, click the "Add Block" button and search for "Custom HTML".

    <ImageBorder>

### JSX Compatibility

If you're using [JSX](https://reactjs.org/docs/introducing-jsx.html), you just need to convert the style attribute of the embed code to an object:

<div>Before:</div>

```
style="border-radius: 10px; height: 57px; width: 229px;"
```

<div>After:</div>

```
style={{
    borderRadius: "10px",
    height: "57px",
    width: "229px"
}}
```
